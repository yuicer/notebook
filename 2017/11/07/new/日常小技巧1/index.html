网页截屏

html2canvas 可以将某个 dom 转化成 canvas，再转为图片

但是它作用有很多限制，对于写在 style 中的 css 样式可能会不生效，必须加到 dom 上 style=””，另外对于部分 css 不支持
而且不能实现截屏，只能选中某个 dom

打印输出改变样式 %c 后的部分样式会被改变

console.log(‘%c red’,’color: red; font-size:50px’)

url 格式
