### 6
传回调函数去获取异步函数的值
```
var url;
function callback(url_) {
	url = url_;
}
yibu(callback)
//异步函数中
callback(url);
```
### 6
webpack 中如果想要引用入口文件，设置如下
```
entry: {
  a: "./a", // a requires b
  b: ["./b"] // workaround: "./b" can now be in another bundle
}
```
这样就可以直接去引用 b 入口文件，加[]就好
